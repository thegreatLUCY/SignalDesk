# Screenshots

Drop PNGs here with these exact filenames — the main README references them.
(Claude can't capture the running browser UI, so these are taken by hand.)

| filename | what to capture |
|---|---|
| `dashboard.png` | Full dashboard: sidebar + one open chart with overlays. |
| `briefing.png` | Daily Briefing bar **expanded** — risk chip + provenance badge + body. |
| `watchlist.png` | The sidebar alone: grouped Equities/Crypto, price + % per row. |
| `signals.png` | A chart with the legend bar visible (overlays on, MA/Bollinger lines). |
| `compare.png` | Two charts side-by-side (compare mode, `?a=…&b=…`). |
| `annotation.png` | Briefing bar showing a `tier 2 · codex/claude` annotation card. |

Recommended: ~1600px wide, PNG. After adding them:
`git add docs/screenshots && git commit -m "docs: add screenshots"`
